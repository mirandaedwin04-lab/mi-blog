Busco reducir tareas repetitivas. Hasta hoy, subir mi blog requeria escribir tres comandos de Git manualmente cada vez.

Hoy creé un script en Python (`publicar.py`) que:
1. Detecta la fecha y la hora actual.
2. Añade todos los cambios.
3. Hace un commit con un mensaje automatico.
4. Empuja los cambios a GitHub/Netlify.

¡Ahora actualizo mi blog con un solo doble clic!
