Hoy aprendi una leccion valiosa **"No se trabaja directo en Producción"**. Aprendí a configurar mi entorno local y le di una identidad visual profesional al blog.

## 1. El entorno Local (Localhost)
Descubrí que cada vez que subia un cambio en la nube, gastaba recursos de Netlify. Para optimizar mi flujo de trabajo, ahora ahora levanto el servidor de forma local.
El comando mágico para ver mi blog sin internet y probar cambios rápido:
```bash
pelican content -r -l
```
