Hoy comence mi camino creando mi propio blog profesional. No fue facil, pero aprendi mucho sobre como funcionan las páginas web por detrás.

### ¿Que herramientas use?
* **VS Code:** Mi editor de código.
* **Pelican:** El motor de python que crea el blog.
* **Git:** Para guardar las versiones de mi trabajo.
* **Netlify** Para publicar la página en internet.

### Los retos que superé
Al principio tuve vaios errores. El mas dificil fue el **Error 127** en Netlify. Ocurrio porque Linux no encontraba el comando `pelican`

**¿La Solución?**
Tuve que cambiar el comando de contruccion a ```python -m pelican content``` para obligar al sistema a encontrar la libreria.

¡Ahora mi blog es visible en todo el mundo!
